Info-Blatt für den Tester

  • Für Sie ist noch kein Benutzer erstellt. Nutzen Sie die Maske links um einen Nutzer zu erstellen.
  • Es gibt "normale" Nutzer und Admin Nutzer. Um "aufsteigen" zu können, benötigen Sie einen Token wie z.B.: ABC-DEF-GHI
  • Wetterdaten sind in der Regel vom 01.01.2017 bis heute+14 Tage vorhanden
  • Für die Orte Regensburg und Landshut gibt es im Zeitraum von 01.01.2018 - 30.04.2018 einige Veranstaltungen als Testdaten (können natürlich auch selbst generiert werden)
  • Diese Informationen sind stets im Menüpunkt Startseite verfügbar
